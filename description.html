Error Analysis

Analysis of Errors Used in Our Project

This documentation provides a detailed overview of error analysis, numerical methods, and tools for exploring approximations and errors in numerical computations.

Features

1. Absolute Error
2. Relative Error
3. Percentage Error
4. Truncation Error
5. Round-off Error
6. Error Propagation Analysis

1. Absolute Error: The absolute difference between the true value and the approximate value. Formula: Ea = |Xt - Xa|

2. Relative Error: The normalized absolute error, defined as: Er = |Ea| / |Xt|

3. Percentage Error: The relative error multiplied by 100. Formula: Ep = Er * 100

4. Truncation Error: Errors introduced when infinite processes (e.g., Taylor series) are approximated with a finite number of terms.

5. Round-off Error: Errors introduced when numbers are rounded during numerical computation, often due to limitations in floating-point representation.

6. Error Propagation: The effect of individual errors propagating through arithmetic operations in computations. Includes addition, subtraction, multiplication, and division.

Error Propagation Analysis

Addition/Subtraction: Total error is the sum of absolute errors of the operands. Formula: Ex+y = Ex + Ey

Multiplication: Total error depends on relative errors of operands. Formula: Exy = |XxYy|(|Ex/Xx| + |Ey/Yy|)

Division: Error depends on relative errors of numerator and denominator. Formula: Ex/y = |Xx/Yy|(|Ex/Xx| + |Ey/Yy|)

Conclusion

Understanding error analysis is vital in numerical methods. This tool provides a robust way to quantify and analyze errors, ensuring the reliability of numerical computations.

Error Examples

Absolute Error: Suppose the true value is 2.11, and the approximate value is 2.10. Then: 

          Ea = |2.11 - 2.10| = 0.01

Relative Error: Using the same values, the relative error is: 

          Er = Ea / |Xt| = 0.01 / 2.11 = 0.0047

Percentage Error: Multiply the relative error by 100: 

          Ep = Er * 100 = 0.47%
